

# Snapclip — Data to Media Agent | Hackathon 2026

A multi-page web application that showcases an AI-powered service to transform static files (PDFs, Excel, Images) into shareable videos. Uses the Snapclip logo and a modern purple/blue gradient color scheme inspired by the reference images.

---

## Page 1: Landing Page

### Hero Banner
- Snapclip logo prominently displayed
- Headline: **"Data to Media Agent"** with subtitle **"Hackathon 2026"**
- Bold CTA button: **"Transform Your Files Into Videos"**
- Animated gradient background in purple-to-blue tones

### Service Description
- Short section explaining the core value: turn complex data into illustrative video content with one click
- 4 feature cards (inspired by reference images): Fast Delivery, Convert Existing Materials, Effortless Updates, Multi-Format Output

### 5 FAQs
- Accordion-style FAQ section covering common questions about the service

### Testimonials
- 3-4 colleague testimonial cards with names, roles, and quotes about the service

### Training & Onboarding
- Section describing onboarding support and training services available

### Pricing
- Freemium tier highlighted with a smiley badge: "Available for Hackathon Supporters 😊"
- Pro and Enterprise tiers shown

### Outlook / Coming Soon
- Feature preview: Full Agentic Automation, Video Production MCP, Sustainable AI Usage via efficient prompting

---

## Page 2: File Upload & Inspiration Sources

### File Upload Interface
- Drag-and-drop zone accepting PDF, Excel, and Image formats
- File list with icons showing uploaded files
- Section for **Inspiration Sources**: input field to paste YouTube URLs for style reference

---

## Page 3: Video Configurator

### Configuration Form with these sections:

**Sensitivity & Visibility**
- Sensitivity: Public, Restricted, Secret
- Audience: Public or Internal

**Video Settings**
- Length slider: 15 seconds to 30 minutes
- Mode: Single Video or Video Compilation (with amount selector)

**Content Brief**
- Content Description textarea
- Target Audience selector
- Core Messages & Objectives
- Publication Time: Start and End date pickers

**Creator & Style**
- Select Video Creator: 3 example names (e.g., Alex Studio, Maya Productions, Leo Creative)
- Video Style: Educational, Promotion, Branding
- Tonality: Educational, Corporate, Conversational

**Output Configuration**
- Format: Landscape, Vertical
- Channels: YouTube, WhatsApp, MS Streams, LinkedIn, Instagram, X (multi-select)
- Accessibility: text field for disability accommodations (sound, colors)

**Transparency Section**
- Live token/cost calculation display
- CO₂ impact statement: "This 3-minute video (4 formats) generated approx. 0.5 kg CO₂, fully offset, delivered via green streaming"

**CTA**: "Generate Magic Prompt" button
